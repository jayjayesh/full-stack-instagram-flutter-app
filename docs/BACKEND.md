# Backend Guide

The backend lives in `backend/` and uses NestJS.

Start here:

1. Read `docs/LESSON_PLAN.md`.
2. Follow `docs/SETUP.md`.
3. Keep `docs/API.md` open while testing routes.

The most important backend files are:

- `backend/src/auth`: signup, login, JWT, current user.
- `backend/src/posts`: posts, likes, comments, uploads.
- `backend/prisma/schema.prisma`: database tables and relationships.
